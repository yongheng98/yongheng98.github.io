hexo主题搭建个人博客

准备工作

首先，你需要一个github账号，下载node.js，git，后期网站会放到github上。

第一步：

安装hexo：

新建一个文件夹，我的叫blog，建好以后进去。
右键，git bash here，输入

npm -v

看会不会显示版本号，如果显示万事大吉，如果不显示，卸掉node.js，再装一遍（全选默认，装在c盘）。

我第一次碰到的坑在这里，node.js默认装在C盘，我改到F盘，结果就是显示不出来版本号。重启，改环

境变量都不好使，还是卸掉重装的，装在C盘就好了。

显示版本号之后，再输入

npm -v ，

前面一步成功这一步应该也成功了。

接下来就是安装cnpm，之后要用它装hexo。还是在这个窗口，
输入

npm install -g cnpm –registry=https://registry.npm.taobao.org/

这里选择镜像到淘宝因为cnpm在国内的下载速度不友好,安好后同样用

cnpm -v 检查,

出来相关信息则成功了。

接着就是hexo了，输入

cnpm install -g hexo-cli,

安装好后老套路

hexo -v检查,

至此前期软件准备算是告一段落。

第二步：

本地搭建：

还是在你新建的文件夹下，打开git bash here，

输入

hexo init

成功之后，最后一行会有提示，

INFO Start blogging with Hexo!

这是第二个坑，我大部分时间，都浪费在这里。他的速度很慢不要急，我是在cmd里进到这个文件里，再用这

个命令，速度就比较快了，不知道为啥。

hexo init成功之后应该会在文件夹下生成8个文件。

如果这一步成功了，就再输入

hexo s，

这一步如果成功会有提示，

INFO Hexo is running at http://localhost:4000

如果有提示就说明这一步成功了。

在浏览器输入

localhost:4000

就能在本地查看你的博客了。

第三步：

部署至github：

新建一个仓库，即new repository，仓库的名称开头一定要是你自己的昵称！！！这个仓库就是存放新建博

客数据的地方。

还是新建文件夹下的git bash here，输入

cnpm install –save hexo-deployer-git

这个插件相当于是沟通你本地数据与github仓库的通道,之后在每次本地数据编辑完后都要用到它,现在有了

通道,本地数据,还有部署的仓库,只差怎样部署了.

至于怎么部署就到文件夹根目录下找config.yml的配置文件进去更改数据形成部署,编译器的话notepad++ ,sublime都行,我用的前者,打开后来到最底部找到deploy:

type后加git 并添加repo: https://github.com/youname/yourname.git/ ; branch: master repo后

面的网址就是新建仓库的网址。

最后，在git bash里输入

hexo d，

会有提示输入github的账号和密码，正常执行完毕之后会出现

INFO Deploy done:git ,

表明部署成功。这时你新建的仓库下会多出许多文件。在浏览器输入你的博客网址，可能会有点慢，刷新等

待即可。至此，博客已经部署到github上了，快邀请你的小伙伴一起来浏览吧！（这一步，网上有人用的密

匙SSH，具体方法请自行百度，密匙还是挺好用的。）
